# ユーザーマニュアル（一般社員向け）

Ryohi-Tax Saver を使うすべての社員・出張者向けのマニュアルです。

---

## 1. 初期設定（初回のみ）

1. App Store / Google Play で **「Ryohi-Tax Saver」** をインストール
2. アプリを開き「**新規登録**」から会社のメールアドレスで登録
3. 会社の管理者から招待メールが届くので、リンクをタップ
4. 設定タブで **氏名・役職** を入力

---

## 2. 出張の記録（基本フロー）

### 🚶 出発

1. 「出張記録」タブを開く
2. 大きな **「出発する」** ボタンをタップ
3. 位置情報の利用を許可
4. 事務所を出るタイミングでボタンを押すのが精度高い

### 🚗 移動中

- アプリは自動で移動距離・経路・時間を記録します
- 画面は閉じても構いません（iOSはバックグラウンドで継続記録）
- バッテリーは通常1時間あたり5〜10%消費

### 🎯 到着

1. 目的地に着いたら「**到着・記録終了**」
2. 到着地の住所が自動取得されます
3. 以下を入力:
   - **出張目的**（必須）例: 取引先との打ち合わせ
   - **訪問先（企業名）** 例: 株式会社サンプル
   - **面会相手** 例: 営業部 山田様
   - **成果・報告** 例: 新製品の見積提出、次回4月末
   - **交通手段**（車・電車・飛行機など）
   - **宿泊数**（宿泊した場合は泊数を選択）
4. 「**記録を保存**」をタップ

---

## 3. 経費（立替精算）を登録

1. 「経費」タブ → 「**経費を追加**」
2. 領収書を **カメラで撮影** or 写真ライブラリから選択
3. AIが金額・取引先・登録番号を自動抽出（Proプラン以上）
4. 必要に応じて修正:
   - 日付 / カテゴリ（交通費・宿泊費・会議費等）
   - 金額（税込）
   - 税区分（課税10% / 軽減8% / 不課税 等）
   - 適格請求書発行事業者登録番号（T+13桁）
   - 支払方法（立替 / 法人カード / 現金 / ICカード）
5. 「**保存**」

---

## 4. 申請と承認

1. 記録した出張カードの ✈️ **「送信」アイコン**をタップ
2. 上長に自動で通知が飛びます
3. 状態は **ドラフト → 申請中 → 承認済** と進みます
4. 差戻しされた場合は理由が表示されるので修正して再送信

---

## 5. 精算書の出力

1. 「精算書」タブで **月を選択**
2. 「**精算書を生成**」→ PDFが作成されます
3. 「共有」で上長や経理にメール送信 or AirDrop で受け渡し

各出張カードの **PDFアイコン**からは、GPS軌跡入りの **出張報告書** が個別に出力できます。

---

## 6. よくある操作

| やりたいこと | 操作 |
|---|---|
| ログアウト | 設定タブ → ログアウト |
| 別端末へ引っ越し | クラウド同期が有効なら、新端末でログインするだけ |
| 出張記録を削除 | 出張カードのゴミ箱アイコン |
| 宿泊数を変更 | 到着時のモーダルで 1〜5泊を選択 |

---

## 7. トラブル対処

| 症状 | 対処 |
|---|---|
| GPS が記録されない | 位置情報の許可を確認。「常に許可」推奨 |
| 距離が少なく出る | 通路側の飛行機内・地下鉄・トンネルなどでは不安定 |
| 保存ボタンが押せない | 出張目的が未入力 |
| ログインできない | パスワードリセットを試す。それでもダメなら会社管理者に依頼 |
